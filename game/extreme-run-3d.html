About This Game

Ever find yourself so immersed in a game that you start moving along with your character? Extreme Run 3D delivers this intense experience!

Extreme Run 3D is a fast-paced running game where you control a rolling ball through 11 challenging levels with VR-like immersion.

Controls

  • ← → Keys: Move
  • ↑/Space: Jump
  • C: Change view
  • H: Hide UI

Gameplay

  • 4 difficulties: Easy to Extreme
  • 44 total levels with increasing challenge
  • Master obstacle dodging to beat all levels

Level Highlights

Level 1-3:

Learn controls, avoid spikes & invisible walls

Level 4-6:

Rotating platforms, deadly hammers & lasers

Level 7-11:

Tunnel runs, ghost battles & ultimate challenges
